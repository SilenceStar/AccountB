# AccountBook：《账号册子》
> 一个在本地记录、保存账号密码以及证件照的App，支持指纹解锁。
> 欢迎各位Fork以及提交MR
> 如果对此App有好的idea，欢迎提issue 😁


（App最近更新时间：2019/11/19）/ Android Studio 3.5.2 /

* 支持随机密码生成、
* 支持账号密码的备份和导入
* 支持保存以及查看证件照
* 密码采用AES对称加密
